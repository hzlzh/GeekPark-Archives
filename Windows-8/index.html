Windows 8 完全手册

关于此手册：

崭新的用户交互、崭新的平台战略，无论成败，Windows 8 必将成为微软 Windows 史上最大的变革和冒险。本手册将为您扫清迷雾，全面解读 Windows 8 的特性与体验。从 Metro UI 与传统桌面的杂合到触摸优化的操作体验，到移动为中心的功能体验，以及更多大大小小的功能特性，我们将解析微软新一代操作系统 Windows 8 背后的故事。

关于作者

董 超- Afio
极客公园资深编辑，Windows 深度用户。全程关注 Windows 8 DP 版、CP 版、RP 版、RTM 版、正式版的进化历程，对 Windows 8 有深刻的见解。
